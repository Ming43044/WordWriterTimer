# 截图文件夹

这个文件夹用来存放 README.md 中引用的截图。

## 需要添加的截图

请按以下文件名上传截图，README.md 中已经预留了对应的位置：

| 文件名 | 内容说明 | 拍摄时机 |
|--------|---------|---------|
| `status-bar-preview.png` | Word 底部状态栏显示统计信息 | 运行 `StartWriting` 后打字时 |
| `vba-editor.png` | VBA 编辑器整体界面 | 导入 `WordWriterTimer.bas` 后 |
| `import-module.png` | 左侧工程资源管理器显示 `WordWriterTimer` 模块 | 导入成功后 |
| `form-design.png` | 窗体设计界面，6 个 Label 排列好 | 创建窗体时 |
| `enable-macros.png` | 信任中心 → 宏设置界面 | 启用宏时 |
| `run-macro.png` | Alt+F8 宏对话框 | 运行宏时 |
| `quick-access-toolbar.png` | Word 顶部快速访问工具栏 | 添加按钮后 |
| `floating-window.png` | 浮动统计窗口 | 运行 `ShowStats` 后 |

## 如何添加截图

1. 在 Word 中按 **Print Screen** 或 **Win + Shift + S** 截图。
2. 保存为 PNG 格式。
3. 把图片文件放到这个 `images/` 文件夹里。
4. 确保文件名与上表一致。

## 注意事项

- 截图中不要包含敏感文档内容。
- 建议只截取界面关键部分，不要整张屏幕截图。
- 如果某个截图暂时不想加，可以保留占位符，不影响 README 的阅读。
